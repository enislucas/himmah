▲ Switch on the AI

Every step from "AI is off" to "the AI helpers are live and tested", with what you should see after each step and what to do if you don't. Tick the boxes as you go - they're remembered on this PC. You should not need to ask "now what?" at any point - and if you get stuck, you can always just skip this guide entirely and keep using Himmah without AI.

Before you start, know this: Himmah already works 100% without this - capture, the board, the calendar, habits, rhythms, Naseer and the guided tour all run fully offline. This guide only switches on the optional AI helpers: ✦ Enhance and ✦ Enrich (sharpen a messy capture), Inbox Triage, Calendarize tomorrow/my week, the Idea Parking Lot's Sort/Enhance, and the Make it yours onboarding chat. Total time: about 2 minutes on Himmah's side (Part 2-3 below); the DeepSeek signup in Part 1 is a one-time thing that usually takes another 3-5 minutes the first time. Cost: DeepSeek's own published price for the model Himmah uses is about $0.14 per million input tokens and $0.28 per million output tokens - and on top of that, Himmah enforces its own hard 10¢/day cap on the server, so it can never run away no matter what. New DeepSeek accounts often get a small free trial balance too, so your first tests may cost nothing at all.

Part 1 - Create your DeepSeek account & key (3-5 min, one time ever)

1. Sign up at the API platform (not the chat app) Go to platform.deepseek.com and sign up with your email (some accounts can also use a phone number). No credit card is required just to create the account.
You should see: the DeepSeek Platform dashboard (usage graphs, a left-side menu with items like API keys and Top up / Billing) - not the consumer chat site.
If you land on a chat window instead: that's chat.deepseek.com, the wrong site. Check the address bar says platform.deepseek.com.
2. Check your balance (top up only if you need to) Open Top up (sometimes called Billing) in the left menu. New accounts sometimes start with a small free granted balance for testing - if you already see a non-zero balance, you can skip paying anything for now. When it runs low, add the smallest amount offered (a few dollars); at DeepSeek's own per-token price plus Himmah's 10¢/day cap, that lasts weeks to months of normal use.
You should see: a non-zero balance shown on the dashboard (free trial credit or your own top-up).
If a brand-new key errors immediately with a balance/payment message even though you see free credit: some regions require a card on file before any key works at all. Add a small top-up and try again.
3. Create an API key and copy it immediately Left menu → API keys → Create new API key → name it himmah → Create. A key starting with sk- appears - DeepSeek shows the full key only this one time. Click copy and paste it into a temporary Notepad right away so you can't lose it before the next step.
You should have: a string like sk-xxxxxxxxxxxxxxxxxxxxxxxx sitting in a Notepad window.
If you closed the popup before copying: you can't reveal it again - no problem, delete that key on the same page and create a fresh one. Keys are free and unlimited.

Part 2 - Give the key to Himmah (1 min)

4. Find the key file In the folder you extracted Himmah into, open the data sub-folder. There is a file called deepseek_key.txt (it ships empty, 0 bytes) sitting next to tasks.json. Full path from the Himmah folder: data\deepseek_key.txt - there is no separate "app" folder in this build, just data directly inside the Himmah folder.
You should see: an empty deepseek_key.txt (0 KB) inside data.
If you don't see a .txt ending: Windows hides known file extensions by default. In File Explorer's View menu, tick File name extensions so you can see exactly what each file is called (and avoid accidentally saving it as deepseek_key.txt.txt later).
5. (Free rehearsal) Paste the word MOCK and test the plumbing first Open deepseek_key.txt in Notepad, type exactly MOCK (4 capital letters, nothing else, no extra line), and Save. This runs every AI feature in a free, fully offline "pretend" mode - it proves the file is in the right place and the right format before you spend a single cent. This is the safest way to learn the flow at $0.
You should see (after step 7's reload): clicking the ✦ AI button shows on · mock mode (no cost) with a purple dot, and running ✦ Enhance on a task still tidies it (offline, instant, free) - even though the mock reply itself won't literally say "(mock)" in the text, the AI popup's mock mode label is the real proof you're in rehearsal mode, not a live key.
If the ✦ AI popup still says "off - no key yet" after reloading: the file's location or format is wrong, not the key. Re-check you edited data\deepseek_key.txt inside the Himmah folder, and that it isn't secretly named deepseek_key.txt.txt.
6. Paste your real key Once MOCK works, open the same file again, delete MOCK, paste your real sk-... key (just the key - no quotes, no spaces, no extra blank lines), and Save.
You should see: the file's size jump from 0 bytes to roughly 35-40 bytes (one short line).
If Notepad offers a "Save As" dialog: keep the file name exactly deepseek_key.txt and set Encoding: UTF-8, Save as type: All files - so it can't quietly become deepseek_key.txt.txt.

Part 3 - Prove it works for real (1 min)

7. Reload Himmah and check the chip Back in the Himmah window, press Ctrl+R to reload. You do not need to restart the server or close the console window - the key file is read fresh on every single AI call, so a reload is always enough.
You should see: the ✦ AI button (top-left of the bar) turns on; clicking it shows on · deepseek-v4-flash with a purple dot and a spend line reading 0.00¢ of 10¢ today · 0 calls · hard cap, enforced by the server.
If it still reads "off - no key yet": the file is empty, misplaced, or misnamed - redo step 4-6. Note: this indicator lights for any non-empty file, even a wrong key - so don't trust the chip alone; step 8 is the real proof.
8. The real test: run ✦ Enhance on a task Type any messy thought into the top capture bar (e.g. gtta email prof abt thesis tmrw) and press Enter. It lands in the Inbox (top bar). Open the Inbox, find your new item, and click its ✦ Enhance button.
You should see: a preview popup with the original on one side and a cleaned-up rewrite on the other, with ✓ Use enhanced / ✗ Keep original buttons - approve or reject, nothing is ever applied silently. Then reopen the ✦ AI button: the spend line should now read something tiny but non-zero, like 0.01¢ of 10¢ today · 1 call - that non-zero number (never seen in mock mode, which always stays 0.00¢) is the real, unambiguous proof your key is live and DeepSeek was actually billed.
"⚠ DeepSeek HTTP 401 - ..." toast: the key is wrong or was revoked - create a fresh one (Part 1, step 3) and redo Part 2. "⚠ daily AI budget (10¢) reached...": you've hit today's hard cap; it resets at midnight, nothing is broken. "⚠ DeepSeek unreachable (...)": you're offline right now - AI needs internet, the rest of Himmah doesn't. "✦ already clean - kept as-is": that's fine - the AI decided your text needed no changes, the guardrail working as designed.
